⚠  brms Intercept-Prior — Hinweis
Bei Formeln wie y ~ x oder y ~ 1 + x nimmt brms intern an, dass alle Prädiktoren zentriert sind, und setzt den Intercept-Prior automatisch — eigene Prior-Angaben für das Intercept werden dabei ignoriert.

Soll das Intercept mit einem eigenen Prior versehen werden, muss es explizit modelliert werden: y ~ 0 + Intercept + x
Nur in dieser Formulierung greifen eigene Intercept-Priors wie gewünscht. → Prädiktoren vor der Analyse zentrieren wird dringend empfohlen.
